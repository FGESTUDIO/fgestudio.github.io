FGES 网站内容后台

免费安全版：编辑后导出 content.json，再上传到 GitHub。

基础资料

这里会影响网站的公司名、WhatsApp、Email 和社交媒体链接。

写国家代码，不要加 +。例如 60172599400。

价格配套

这里只改价格。配套名称、项目说明可以到“全部文字”里搜索 pricing 修改。

作品图片

上架图片的做法：先把图片文件上传到 GitHub 的 images 文件夹，然后在这里填写路径， 例如 images/poster-01.jpg。选择本地图片只会帮你自动填文件名和预览，不会真正上传图片。

全部文字

这里可以改网站几乎所有文字。先选语言，再搜索关键词，例如 hero、services、pricing、portfolio、mcn、contact。

导出与上传

修改完成后，点击“下载 content.json”。然后到 GitHub 仓库上传并覆盖根目录的 content.json。 如果你有新增图片，也要把图片上传到 images 文件夹。等 GitHub Pages 部署成功后，网站就会更新。
